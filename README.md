## ecommercio

ecommercio is a crawler that deep crawls various shopsystem enviroments and extracts all products into a separated database.

## Supported Shopsystems

- Shopware 5

---

## Planned Shopsystems

- WooCommerce 3.8.1
- Magento 2.3.2
- Shopify
- Plentymarkets

## Usage

Install dependencies:

```
npm install
```

Run the app:

```
npm start
```
